# 隐私说明

xPoster 设计为本地浏览器扩展。

## 它会读取什么

- 你粘贴或导入到侧边栏的 Markdown 文本。
- 当前 X/Twitter 标签页 URL 和页面状态。
- 只有在你选择本地文件夹后，才读取本地图片文件。
- 只有在 Chrome 授权图片网站后，才读取网页图片文件。

## 它会保存什么

- 草稿文本会保存在 Chrome 扩展本地存储中，方便你重新打开侧边栏时恢复。
- 如果你选择了本地图片文件夹，会保存对应的文件夹授权句柄。
- 检查和导入记录会显示在侧边栏中。

## 它不会做什么

- 不发送分析统计。
- 不使用后台服务器。
- 不要求注册 xPoster 账号。
- 不包含许可证验证、订阅或试用限制。
- 不点击 X 的发布按钮。

## 权限说明

- `storage`：在这个浏览器里保存草稿和本地设置。
- `sidePanel`：显示发布控制台。
- `tabs`：找到并检查当前 X Article 标签页。
- `https://x.com/*` 和 `https://twitter.com/*`：在 X/Twitter 页面上运行导入器。
- 已发布版本中的可选图片网站权限：当 Markdown 引用网页图片时，在你允许后读取图片文件。
